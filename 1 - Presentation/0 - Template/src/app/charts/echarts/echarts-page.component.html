Ngx-echarts

ngx-echarts is an open source Angular directive for echarts. We really love this library because it allows you to customize every single detail from every chart. Also, its documentation is super complete and clear.
echarts is written in pure JavaScript and based on zrender, which is a whole new lightweight canvas library.
For more info please visit the plugin's Demo Page or Github Repo.
