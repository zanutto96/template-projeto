Ngx-echarts

ngx-charts is an open source declarative Charting Framework for Angular. It don't merely wrap d3, it uses Angular to render and animate the SVG elements with all of its binding and speed goodness, and uses d3 for the excellent math functions, scales, axis and shape generators, etc.
For more info please visit the plugin's Demo Page or Github Repo.
