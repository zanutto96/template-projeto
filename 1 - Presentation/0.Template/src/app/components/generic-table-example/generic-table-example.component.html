Generic Table Component

Reusable table component with filtering, sorting, and pagination

Overview

The GenericTableComponent is a highly customizable and reusable table component that can be used across your entire application. It supports multiple column types, filtering, sorting, pagination, and row selection.

Features:

  • ✅ Multiple column types (text, numeric, currency, date, boolean, checkbox, progress, thumbnail, actions)
  • ✅ Global and per-column filtering
  • ✅ Column sorting
  • ✅ Pagination with customizable page sizes
  • ✅ Row selection (single/multiple)
  • ✅ Action buttons per row
  • ✅ Fully responsive design

Example 1: Basic Table

A simple table with text and numeric columns, filtering and pagination.

Usage:

<app-generic-table
  [data]="basicTableData"
  [config]="basicTableConfig">
</app-generic-table>

Example 2: Advanced Table with All Column Types

Demonstrates all available column types including currency, date, progress, boolean, thumbnail, and actions.

Configuration Example:

columns: [
  {{ '{' }} field: 'id', header: 'Order ID', type: 'text', sortable: true },
  {{ '{' }} field: 'thumbnail', header: 'Image', type: 'thumbnail', config: {{ '{' }} aspectRatio: {{ '{' }}w: 1, h: 1} } },
  {{ '{' }} field: 'product', header: 'Product', type: 'text', sortable: true },
  {{ '{' }} field: 'price', header: 'Price', type: 'currency', sortable: true, 
    config: {{ '{' }} currencySymbol: '$', currencyCode: 'USD' } },
  {{ '{' }} field: 'date', header: 'Order Date', type: 'date', sortable: true },
  {{ '{' }} field: 'progress', header: 'Progress', type: 'progress', sortable: true,
    config: {{ '{' }} showLabel: true, progressColor: '#035388' } },
  {{ '{' }} field: 'active', header: 'Active', type: 'boolean', sortable: true },
  {{ '{' }} field: 'actions', header: 'Actions', type: 'actions',
    config: {{ '{' }} actions: [
      {{ '{' }} id: 'edit', icon: 'fas fa-edit', tooltip: 'Edit', cssClass: 'btn-outline-primary' },
      {{ '{' }} id: 'delete', icon: 'fas fa-trash', tooltip: 'Delete', cssClass: 'btn-outline-danger' }
    ] }
  }
]

API Documentation

TableConfig Interface

Property Type Default Description
columns TableColumn[] required Array of column configurations
paginate boolean true Enable/disable pagination
pageSize number 10 Initial page size
pageSizeOptions number[] [5, 10, 25, 50, 100] Available page size options
showFilters boolean true Show/hide filter section
filtersVisible boolean false Initial visibility of filters
selectable boolean false Enable row selection
cssClass string - Custom CSS classes

TableColumn Interface

Property Type Default Description
field string required Field name in data object
header string required Column header text
type ColumnType required Column type (see types below)
sortable boolean true Enable column sorting
filterable boolean true Show filter for column
width string - Column width (e.g., '100px', '20%')
cssClass string - Custom CSS classes for cells
config object - Type-specific configuration

Column Types

Type Description Config Options
text Plain text display -
numeric Formatted number -
currency Formatted currency value currencySymbol, currencyCode
date Formatted date dateFormat
checkbox Disabled checkbox (read-only) -
boolean Yes/No badge -
progress Progress bar showLabel, progressColor
thumbnail Image thumbnail aspectRatio
actions Action buttons actions[]

Events

Event Payload Description
(actionClicked) {{ '{' }} action: string, row: any } Emitted when an action button is clicked
(rowClicked) any Emitted when a row is clicked
(selectionChanged) any[] Emitted when row selection changes
(pageChanged) PageEvent Emitted when page changes
